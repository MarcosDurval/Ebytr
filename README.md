# Ebytr
O Etrybr é um todo list onde é possivel realizar as ações do CRUD



# Como executar localmente

### Inicializando o APP
execute:
```
docker-compose up -d
```
### Rode os testes do backend:
```
docker exec -it backend npm run test
```

### Parando a execução
execute:
```
docker-compose down --remove-orphans
```
